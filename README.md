# Google Cloud VPC Networking Lab

## Overview
This project demonstrates creating a custom VPC network in Google Cloud Platform, launching a VM instance inside the subnet, and verifying external connectivity.

## Services Used
- Google Cloud VPC
- Compute Engine
- Debian Linux VM

## Steps
1. Created a custom VPC network
2. Created subnet `10.0.0.0/18`
3. Enabled Private Google Access
4. Created VM instance
5. Connected using SSH
6. Tested connectivity using:

ping 8.8.8.8

Successfully verified internet connectivity from the VM instance.
